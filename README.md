# Inn Keeper Sim
Made by Tuna Mert İyiarı and İlhan Güven Erol

### to run the project
```
#r "file path of .dll"
using ConsoleApp1;
Game game = new Game;
game.Start();
```
### to play the game

```
Use 'CallGuest()' to call a guest from the queue.
Use 'GetCookbook()' to see what each item on the menu is, and what they contain.
Use 'GetMenu()' to see which items are available. Available items change regularly.
Use 'ServeDish(string dishName)' to serve a guest their food.
Use 'GetEncyclopedia()' to see the preferences of Guests.
If a Guest waits too long in line to be called in, they may get frustrated.
If they get frustrated, you will hear a beep. That is your cue to hurry up!
```
